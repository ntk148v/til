# OpenStack

OpenStack is an open-source cloud computing platform for building and managing public and private clouds.

## Projects

### Compute & Orchestration

- [Heat](./openstack-heat/) - Orchestration service, autoscaling
- [Magnum](./openstack-magnum/) - Container orchestration engine (Kubernetes, Swarm)
- [Masakari](./openstack-masakari/) - Instance high availability
- [Zaqar](./openstack-zaqar/) - Messaging service

### Networking

- [Neutron](./openstack-neutron/) - Networking service, Open vSwitch

### Deployment & Operations

- [Kolla](./openstack-kolla/) - Production-ready deployment using containers
- [Autohealing](./openstack-autohealing/) - Automatic repair of failed instances

### Workflows & Telemetry

- [Mistral](./openstack-mistral/) - Workflow service
- [TaskFlow](./openstack-taskflow/) - Task execution and state management
- [Telemetry](./openstack-telemetry/) - Monitoring and metering
- [Vitrage](./openstack-vitrage/) - Root cause analysis and alerting

### Load Balancing

- [Octavia LBaaS](./openstack-octavia-lbaas/) - Load balancer as a service

## Specialized Topics

- [CPU Pinning](./cpupinning/) - NUMA and CPU affinity for VMs
- [VDI](./vdi/) - Virtual Desktop Infrastructure
