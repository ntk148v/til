## 1. Introduction

- Consul is a service mesh solution providing a full featured control plane with service discovery, configuration, and segmentation functionality.
- Main features:
  - Service Discovery
  - Health Checking
  - KV Store
  - Secure Service Communication
  - Multi Datacenter
